## <a name="httpcontext-access-from-a-background-thread"></a><span data-ttu-id="3bf7b-121">백그라운드 스레드에서 HttpContext 액세스</span><span class="sxs-lookup"><span data-stu-id="3bf7b-121">HttpContext access from a background thread</span></span>

> [!IMPORTANT]
> **Azure App Service를 포함한 ASP.NET Core 미리 보기 릴리스**
>
> ASP.NET Core 미리 보기 릴리스는 기본적으로 Azure App Service에 배포되지 않습니다. ASP.NET Core 미리 보기 릴리스를 사용하는 앱을 호스팅하려면 [Azure App Service에 ASP.NET Core 미리 보기 릴리스 배포](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service)를 참조하세요.

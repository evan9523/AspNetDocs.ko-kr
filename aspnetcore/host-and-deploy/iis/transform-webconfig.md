## <a name="build-configuration"></a><span data-ttu-id="b9ff2-113">빌드 구성</span><span class="sxs-lookup"><span data-stu-id="b9ff2-113">Build configuration</span></span>

<span data-ttu-id="b9ff2-116">다음 예제에서는 구성별 환경 변수가 *web.Release.config*에서 설정됩니다.</span><span class="sxs-lookup"><span data-stu-id="b9ff2-116">In the following example, a configuration-specific environment variable is set in *web.Release.config*:</span></span>

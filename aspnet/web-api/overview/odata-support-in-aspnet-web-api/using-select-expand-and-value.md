[Mike Wasson](https://github.com/MikeWasson)

$에 대 한 개요 및 코드 샘플 $select, 확장 및 asp.net Web API 2 OData에서에서 $value 옵션 4.x 합니다. 이러한 옵션에는 서버에서 반환 되는 표현을 제어에 대 한 클라이언트를 수 있습니다.

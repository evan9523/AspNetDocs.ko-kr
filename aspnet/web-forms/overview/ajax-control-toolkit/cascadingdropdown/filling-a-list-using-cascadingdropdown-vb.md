- `ServiceMethod`: 목록 항목을 제공 하는 웹 메서드
- `TargetControlID`: 드롭다운 목록의 ID
- `Category`: 호출 될 때 웹 메서드에 제출 되는 범주 정보
- `PromptText`: 서버에서 목록 데이터를 비동기적으로 로드 하는 경우 표시 되는 텍스트

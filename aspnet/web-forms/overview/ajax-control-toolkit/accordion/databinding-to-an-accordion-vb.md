## <a name="overview"></a>개요

AJAX Control Toolkit의 Accordion 컨트롤 여러 창을 제공 하 고 둘 중 한 번에 표시할 수 있습니다. 패널은 일반적으로 페이지 자체 내에서 선언 하지만 데이터 소스에 바인딩하는 더 많은 유연성을 제공 합니다.

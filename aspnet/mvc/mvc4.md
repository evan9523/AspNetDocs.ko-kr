# <a name="aspnet-mvc-4"></a>ASP.NET MVC 4

- ASP.NET Web API
- 새롭게 정비 되 고 현대화 기본 프로젝트 템플릿

- 모바일 앱을 지원 하기 위해 여러 가지 새로운 기능
- 비동기 메서드에 대 한 향상 된 지원
